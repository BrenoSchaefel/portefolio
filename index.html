03. Projetos em Destaque

ERP de Automação Comercial Web/Desktop

Destaque
PHP Laravel PostgreSQL API REST Sincronização Bidirecional

Sistema completo de automação comercial desenvolvido para operar tanto online quanto offline, atendendo empresas de médio e grande porte do varejo e atacado. O projeto envolveu desde a análise de requisitos, arquitetura, desenvolvimento fullstack, até a implantação e treinamento dos usuários.

  • Versão web para gestão (cadastros, relatórios, NF-e, controle de estoque e financeiro)
  • PDV desktop instalável para locais com internet instável, garantindo operação contínua
  • Sincronização bidirecional via API personalizada, com resolução automática de conflitos
  • Total conformidade com legislação fiscal brasileira, incluindo emissão de notas fiscais eletrônicas
  • Implementação de autenticação segura, controle de permissões e logs de auditoria
  • Desenvolvimento de integrações com sistemas de terceiros (contabilidade, bancos, marketplaces)

Diferencial: Solução híbrida que elimina perda de vendas mesmo sem conexão, com sincronização automática quando a internet volta. Projeto desenvolvido 100% solo, com foco em escalabilidade, segurança e experiência do usuário. O sistema atualmente atende centenas de usuários simultâneos e já processou milhões em transações.

Sistema de Contabilidade Pública

Destaque
Java PostgreSQL Delphi SQL Server Microsserviços

Atuação em sistemas de contabilidade pública utilizados por prefeituras e órgãos governamentais em todo o Brasil. O desafio foi modernizar um sistema legado robusto, garantindo a continuidade dos serviços e a conformidade com normas do setor público.

  • Migração do sistema legado (Delphi 2006 + SQL Server) para nova versão em Java, com arquitetura baseada em microsserviços
  • Desenvolvimento de APIs REST para integração com sistemas governamentais (SIAFIC, eSocial, Receita Federal)
  • Manutenção e evolução das duas versões em paralelo, garantindo transição suave para os clientes
  • Implementação de rotinas de importação/exportação de dados, relatórios fiscais e módulos de prestação de contas
  • Automação de processos manuais, reduzindo erros e aumentando a produtividade dos usuários

Resultado: Transição bem-sucedida para os clientes, mantendo todas funcionalidades do sistema antigo enquanto adiciona novas capacidades. O sistema atende atualmente dezenas de municípios, processando grandes volumes de dados fiscais e contábeis com alta confiabilidade.

ERP de Automação Comercial

Delphi MySQL NF-e PDV Legislação Fiscal

Sistema desktop completo desenvolvido do zero, focado em pequenas e médias empresas do comércio. O projeto envolveu desde a concepção, levantamento de requisitos, desenvolvimento, testes, implantação e suporte contínuo.

  • Módulo fiscal com emissão de NF-e e NFC-e, totalmente integrado à Receita Federal
  • Controle de estoque, vendas, compras e financeiro integrado
  • PDV (Ponto de Venda) otimizado para alta performance e facilidade de uso
  • Implementação de rotinas de backup automático e recuperação de dados
  • Customização de relatórios gerenciais e dashboards para tomada de decisão

Destaque: Desenvolvimento 100% solo, desde análise de requisitos até implantação. O sistema já foi utilizado por dezenas de empresas, contribuindo para a modernização e automação de processos internos.
